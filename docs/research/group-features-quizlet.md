## Quizlet Group Features Analyse

### 1. Classes (Klassen)

Quizlet's kern organisatiestructuur voor groepen.

**Functionaliteit:**
- Docent maakt een "Class" aan
- Studenten worden uitgenodigd via link of code
- Docent kan study sets toewijzen aan de klas
- Alle leden zien dezelfde content

**Kenmerken:**
| Aspect | Details |
|--------|---------|
| Aanmaken | Alleen met Teacher account |
| Lidmaatschap | Studenten moeten ingelogd zijn |
| Content delen | Sets en folders toewijzen |
| Integratie | Google Classroom, Microsoft Teams |

### 2. Quizlet Live (Classroom Game)

Real-time, collaboratieve quiz game voor in de klas.

**Hoe het werkt:**
1. Docent selecteert een study set
2. Genereert een join code
3. Studenten joinen via code (geen account nodig!)
4. Random teams worden gevormd
5. Elke student ziet dezelfde vraag, maar verschillende antwoordopties
6. Teams moeten samenwerken om het juiste antwoord te vinden
7. Fout antwoord → terug naar nul
8. Eerste team met 12 correcte antwoorden wint

**Game Modes:**
- **Teams mode:** Samenwerking, communicatie oefenen
- **Individuals mode:** Solo competitie (min. 2 spelers)

**Vereisten:**
- Minimaal 6 term-definitie paren in set
- Alleen via website starten (niet app)
- Spelers kunnen via app joinen

**Sterke punten:**
- Geen account nodig voor deelnemers
- Werkt op elk device met internet
- QR code om te joinen
- Geschikt voor remote/hybrid learning

### 3. Class Progress (Voortgang Tracking)

Dashboard voor docenten om studieleergedrag te monitoren.

**Wat je ziet:**
- Welke studenten zijn gestart / klaar met studeren
- Hoogste scores op Test, Match, Gravity modes
- Meest gemiste termen (class-level aggregate)
- Tijdsfilters (laatste 2 weken, maand, etc.)

**Data per student:**
- Start/eind tijd per study mode
- Scores per activiteit
- Welke termen lastig zijn

**Data per klas:**
- Aggregate "gemist percentage" per term
- Trends over tijd
- Identificatie van moeilijke concepten

**Beperkingen:**
- Alleen beschikbaar voor Quizlet Plus Teachers
- Studenten moeten ingelogd zijn tijdens studeren
- Niet bedoeld als assessment tool (geen cijfers)
- Alleen via website te bekijken

### 4. Study Groups (Nieuw - Aug 2025)

Peer-to-peer samenwerking feature.

**Functionaliteit:**
- Studenten kunnen vrienden uitnodigen
- Gedeelde voortgang zichtbaar
- Real-time zien wie actief studeert
- Groepsvoortgang tracking

**Verschil met Classes:**
- Classes = top-down (docent → studenten)
- Study Groups = peer-to-peer (studenten onderling)

### 5. Sharing & Permissions

**Delen opties:**
- Directe link naar set
- Email
- Google Classroom
- Microsoft Teams
- Remind (education messaging app)

**Visibility settings:**
- Public (iedereen kan vinden)
- Unlisted (alleen met link)
- Private (alleen uitgenodigde users)
- Classes only (beperkt tot klas leden)

### 6. Quizlet for Schools (Organisatie Tier)

School-brede licentie met volume korting.

**Features:**
- Discounted pricing gebaseerd op aantal licenties
- Admin dashboard voor beheer
- Bulk user management
- Premium features voor alle gebruikers

**Prijsindicatie (individueel):**
| Plan | Prijs/jaar |
|------|-----------|
| Quizlet Teacher | €35.99 |
| Quizlet Plus (student) | €19.99 |
| Quizlet Go (student basis) | €11.99 |

### 7. Quizlet Plus - Individuele Premium Features

Wat krijg je als individuele gebruiker met een betaald Quizlet Plus abonnement.

**Pricing:**
- ~$7.99/maand of ~$35.99/jaar (jaarlijks)
- 7 dagen gratis proefperiode
- Promoties rond back-to-school en Black Friday

**Premium Features:**

| Feature | Beschrijving |
|---------|--------------|
| **Geen advertenties** | Ad-free experience |
| **Offline access** | Sets downloaden voor offline studeren |
| **Custom media** | Eigen afbeeldingen en audio toevoegen |
| **Smart grading** | Herkent synoniemen, niet alleen exacte matches |
| **AI Practice Tests** | Gepersonaliseerde oefentoetsen |
| **Magic Notes** | Upload notities → automatisch flashcards genereren |
| **Memory Score** | Track hoe goed je materiaal onthoudt |
| **Quick Summary** | AI-samenvatting van lange teksten |

**AI Features (detail):**

**Magic Notes:**
- Upload handgeschreven of digitale notities (tot 40.000 karakters)
- Automatisch omzetten naar: flashcards, outlines, practice tests, essay prompts
- Werkt met Google Docs integratie
- Genereert ook gerelateerde concepten

**Q-Chat (AI Tutor):** *(Let op: per juni 2025 niet meer beschikbaar)*
- Gebouwd op OpenAI's ChatGPT API
- Socratische methode: stelt vragen ipv geeft antwoorden
- Modes: "Teach Me", "Quiz Me", "Apply Knowledge", "Practice Sentences"
- Adaptieve vragen gebaseerd op studiehistorie

**Memory Score:**
- Voorspelt hoe goed je materiaal onthoudt
- Toont optimale reviewmomenten
- Helpt bij planning voor toetsen

**Gebruikersfeedback:**
- Kritiek: "Features die vroeger gratis waren zitten nu achter paywall"
- Waardering: Serieuze studenten vinden het de prijs waard
- Casual users blijven vaak bij gratis versie

## Competitor Analyse

### Flashcard Apps - Algemeen

| App | Prijsmodel | Unieke features | Sterk punt |
|-----|-----------|-----------------|------------|
| **Anki** | Gratis (desktop), $25 iOS | FSRS/SM-2, volledig aanpasbaar | Open source, krachtig |
| **Brainscape** | Freemium, $59.99/jaar Pro | Confidence-Based Repetition (1-5 rating) | Professionele content libraries |
| **RemNote** | Freemium | Note-taking + auto flashcards | Kennismanagement |
| **StudySmarter** | ~€5.80/maand | AI-generated flashcards | Collaborative features |
| **Anki Pro** | €5/maand | Keuze uit SRS algoritmes | Moderne UI |
| **Mochi Cards** | Freemium | Markdown support | Developer-friendly |
| **Gizmo** | Betaald | Medische templates | Niche: medical students |
| **Laxu AI** | $4.99/maand | AI flashcards uit PDFs | Automatische creatie |

**Observaties:**
- **AI is de trend**: Magic Notes, auto-generation uit PDFs/notities
- **Anki frustratie**: Krachtig maar "designed in 2006" - complexe setup
- **Freemium dominant**: Bijna allemaal gratis basis + premium features
- **Niche winst**: Gizmo voor medisch, Taalhammer voor talen

### Natuur-specifieke Apps

| App | Focus | Prijsmodel | Leren vs Identificatie |
|-----|-------|-----------|------------------------|
| **BirdID** | Vogels (EU) | Freemium | ✅ Leren (quiz) |
| **Merlin** | Vogels (worldwide) | Gratis | ❌ Alleen identificatie |
| **iNaturalist** | Alle soorten | Gratis | ❌ Alleen identificatie |
| **Seek** | Alle soorten (kids) | Gratis | ⚠️ Gamification, geen echte quiz |
| **Larkwire** | Vogelgeluiden | Betaald | ✅ Leren (quiz games) |
| **Chirp!** | Vogelgeluiden | Betaald | ✅ Leren (quiz games) |
| **ChirpOMatic** | Vogelgeluiden | $4.99 | ❌ Alleen identificatie |

**Merlin Bird ID (Cornell Lab):**
- Meest complete vogel-app: 10.000+ soorten
- Sound ID: 100% correct in tests (vogels)
- **Zwakte**: Geen leer/quiz functionaliteit
- Gratis, non-profit (Cornell Lab)

**BirdID:**
- Europese focus
- Quiz-based learning
- Professionele content
- **Zwakte**: Alleen vogels

**Larkwire / Chirp!:**
- Specifiek voor vogelgeluiden leren
- Game-based approach
- 300+ recordings (Noord-Amerika)
- **Zwakte**: Beperkt tot audio, geen visueel

### Kansen voor Naturae

**Gap in de markt:**

| Concurrent | Wat ze missen |
|------------|---------------|
| Anki | Gebruiksvriendelijkheid, natuur-content |
| Quizlet | Natuur-specifieke integraties (GBIF, xeno-canto) |
| Merlin/iNaturalist | Leer/quiz functionaliteit |
| BirdID | Andere soortengroepen dan vogels |
| Larkwire | Visuele herkenning, alle soortengroepen |

**Naturae's unieke positie:**
1. **Alle soortengroepen** - Niet alleen vogels
2. **Leren + identificatie** - Quiz mode + GBIF integratie
3. **Open content** - Community decks + publieke API data
4. **Moderne UX** - Niet de Anki-complexiteit
5. **Nederlandse markt** - KNNV/IVN samenwerking mogelijk

## Toepassing voor Naturae

### Use Cases Natuurorganisaties

#### 1. KNNV Cursus Soortherkenning

**Scenario:** KNNV afdeling geeft 6-weekse cursus plantenherkenning.

**Gewenste functionaliteit:**
- Cursusleider maakt "Klas" aan voor deelnemers
- Wekelijkse leersets toewijzen (Week 1: Composieten, Week 2: Kruisbloemigen)
- Voortgang per deelnemer zien
- Gezamenlijke quiz tijdens excursie (Quizlet Live-achtig)

#### 2. IVN Natuurgids Opleiding

**Scenario:** IVN leidt vrijwilligers op tot natuurgids.

**Gewenste functionaliteit:**
- Gestructureerde leermodules per soortengroep
- Certificering na voltooien van bepaalde sets
- Docent kan zien wie achterloopt
- Groepsexcursie met live quiz

#### 3. School Biologie Lessen

**Scenario:** Middelbare school, vak biologie.

**Gewenste functionaliteit:**
- Docent maakt klas aan
- Toetsen via quiz mode
- Resultaten exporteren voor beoordeling
- Huiswerk toewijzen

### Database Ontwerp (Conceptueel)

```sql
-- Groepen
CREATE TABLE groups (
  id UUID PRIMARY KEY,
  name TEXT NOT NULL,
  description TEXT,
  organization_name TEXT,
  logo_url TEXT,
  created_by UUID REFERENCES profiles(id),
  invite_code TEXT UNIQUE,
  settings JSONB, -- { requireAccount: boolean, isPublic: boolean }
  created_at TIMESTAMP
);

-- Groepsleden
CREATE TABLE group_members (
  group_id UUID REFERENCES groups(id),
  user_id UUID REFERENCES profiles(id),
  role TEXT DEFAULT 'member', -- 'admin', 'member'
  joined_at TIMESTAMP,
  PRIMARY KEY (group_id, user_id)
);

-- Toegewezen decks
CREATE TABLE group_decks (
  group_id UUID REFERENCES groups(id),
  deck_id UUID REFERENCES decks(id),
  assigned_at TIMESTAMP,
  due_date TIMESTAMP, -- optioneel
  PRIMARY KEY (group_id, deck_id)
);

-- Voortgang tracking (aggregate)
CREATE TABLE group_progress (
  group_id UUID REFERENCES groups(id),
  user_id UUID REFERENCES profiles(id),
  deck_id UUID REFERENCES decks(id),
  cards_studied INTEGER,
  cards_mastered INTEGER,
  study_time_seconds INTEGER,
  last_studied_at TIMESTAMP,
  PRIMARY KEY (group_id, user_id, deck_id)
);
```

## Unieke Kansen voor Naturae

### Differentiatie t.o.v. Quizlet

1. **Natuur-specifieke media**
   - GBIF integratie voor foto's
   - Xeno-canto voor geluiden
   - Quizlet heeft dit niet

2. **Excursie Mode**
   - GPS-gebaseerde quiz tijdens wandeling
   - "Welke soorten kun je hier tegenkomen?"
   - Offline beschikbaar

3. **Seizoensgebonden Leren**
   - Automatische suggesties: "Nu trekvogels leren!"
   - Koppeling met waarneming.nl data

4. **Certificering**
   - Officiële samenwerking met KNNV/IVN
   - Erkende certificaten voor soortenkennis
