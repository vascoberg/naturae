# Industry Golden Standards - Leerapps

> Onderzoek naar best practices van populaire leerapps (januari 2025)

## Duolingo's Gamification Geheimen

[Bron: Orizon Blog](https://www.orizon.co/blog/duolingos-gamification-secrets)

### Impact Metrics

| Feature | Impact |
|---------|--------|
| **Streaks** | 7-dagen streak → 3.6x meer engagement |
| **Streak Freeze** | Churn -21% |
| **iOS Widget** | Commitment +60% |
| **XP Leaderboards** | 40% meer lessen per week |
| **Leagues** | Lescompletie +25% |
| **Daily Quests** | DAU +25% |
| **Badges** | 30% meer kans op course completion |
| **Double XP Weekend** | 50% surge in activiteit |

### Psychologische Principes

1. **Loss Aversion** - Mensen willen hun streak niet verliezen
2. **Variable Rewards** - Onvoorspelbare beloningen houden gebruikers betrokken (slot machine effect)
3. **Social Proof** - Leaderboards en competitie
4. **Progress Visualization** - Duidelijke voortgangsindicatoren

### Notificaties Strategie

- Getimed op wanneer gebruiker meestal leert
- Humor en gentle encouragement (niet agressief)
- Gepersonaliseerd op basis van gedrag
- "Guilt trip" met mascotte (adorably)

### Streak Systeem Details

[Bron: Medium - Streak System Breakdown](https://medium.com/@salamprem49/duolingo-streak-system-detailed-breakdown-design-flow-886f591c953f)

- Visuele cues: bright icons voor actieve streaks, grey voor broken
- Streak Freeze: skip één dag zonder streak te verliezen
- Home screen widget: constant zichtbaar

## Quizlet's Study Modes

[Bron: Quizlet Features](https://quizlet.com/features/flashcards)

### Beschikbare Modi

| Mode | Beschrijving | Type |
|------|--------------|------|
| **Flashcards** | Klassiek flippen, shuffle optie | Self-paced |
| **Learn** | Adaptief: multiple choice → written naarmate je beter wordt | Adaptive |
| **Test** | Simuleer een echt examen | Assessment |
| **Match** | Race tegen de klok, drag & drop | Game |
| **Swipe** | Tinder-style: rechts = ken ik, links = niet | Quick review |

### Key Features

- **Know/Don't know sortering** - Visueel feedback per sessie
- **700+ miljoen publieke sets** - Massive library
- **AI-powered uitleg (Q-Chat)** - ChatGPT integratie
- **Magic Notes** - PDF/handwriting → automatisch flashcards
- **Swipe mode** - Snelle review met gesture-based input

### Pricing Model

- Gratis tier met basis features
- Quizlet Plus: $35.99/jaar voor AI features en Learn mode

### Gebruikersfeedback

**Positief:**
- 98% zegt dat begrip verbeterd is
- "Know/don't know" sortering zeer gewaardeerd
- AI features maken studeren leuker

**Kritiek:**
- Paywall frustreert studenten
- Match/Gravity games minder effectief voor lange termijn
- Vendor lock-in: data export moeilijk

## Anki's Sterke Punten

### Unieke Eigenschappen

- **SM-2 algoritme** - Bewezen spaced repetition
- **Extreem customizable** - HTML/CSS card templates
- **Open source** - Geen vendor lock-in
- **Grote community** - Shared decks ecosystem
- **Cross-platform** - Desktop, web, mobile

### Nadelen

- Steile leercurve
- Gedateerde UI
- Mobile app (iOS) is betaald

## Key Takeaways

1. **Streaks werken** - Implementeer streak tracking met freeze optie
2. **Loss aversion > rewards** - Focus op wat gebruikers kunnen verliezen
3. **Variable rewards** - Onvoorspelbaarheid houdt mensen engaged
4. **Swipe = snelheid** - Laagdrempelige interactie voor quick sessions
5. **Progress moet zichtbaar zijn** - Dashboard met statistieken
6. **Notificaties met persoonlijkheid** - Timing + tone of voice
